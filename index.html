Features
Everything your agent needs to ship a game

Nine skills, eight commands, and four autonomous agents handle the hard parts. Works with OpenClaw and Claude Code.

🦞

OpenClaw Native New

Install via ClawHub or paste the repo URL in chat. Your OpenClaw agent gets 9 game dev skills instantly. Same SKILL.md format, zero config.

🎮

2D + 3D Engines

Phaser 3 for 2D side-scrollers, platformers, and arcade games. Three.js for 3D environments with WebGL rendering. Both use the same architecture patterns.

🎨

Visual Design System

Automated visual polish: gradient skies, particle effects, screen shake, squash/stretch, transitions, parallax, and menu animations. Scores each area 1-5 and fixes them.

🎵

Procedural Audio

Chiptune background music and retro sound effects generated in the browser with Strudel.cc. No audio files needed. BGM for every scene, SFX for every action.

🧪

Automated Testing

Playwright tests for gameplay verification, visual regression screenshots, performance budgets, and accessibility audits. Run headless or watch in the browser.

🏗

Clean Architecture

EventBus pub/sub for decoupled modules. Centralized GameState. Every magic number in Constants.js. No circular dependencies. Same patterns for 2D and 3D.

🚀

One-Command Deploy

Guided GitHub Pages setup with auth walkthrough. Sets base paths, builds, deploys. Future changes auto-deploy with npm run deploy.

🤖

Autonomous Agents

Four agents handle complex workflows: end-to-end game creation, test execution and failure fixing, architecture review, and deployment automation.

🌐

Moltbook Ready New

Deploy a game and post the URL to Moltbook. 770K+ agents on m/gaming are waiting to play it, upvote it, and remix it. Games are content on the agent internet.
